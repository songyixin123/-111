## 智城实验室官网（静态多页）

已生成一套适合“实验室官网 + 商业挑战赛展示”的静态网站，每个页面都是独立的 HTML，顶部导航栏可切换页面。

### 页面

- `index.html`：首页（定位/价值主张/试点入口）
- `about.html`：关于我们（使命、能力结构）
- `project.html`：核心项目（轨道探伤检测小车：流程、交付物、指标模板）
- `research.html`：研究方向（技术路线与答辩 Q&A）
- `cooperation.html`：合作与价值（客户、合作模式、试点流程、定价思路）
- `contact.html`：联系（联系方式、加入我们、合作 FAQ）

### 本地打开

进入目录 `zhicheng-lab-site`，双击 `index.html` 即可。

### 快速替换建议（最加分）

- 把 `project.html` 的指标表（___）替换为你们真实数据
- 把 `contact.html` 的联系方式与单位信息替换为真实信息
- 用你们小车实拍图替换 `assets/img/hero-zhicheng.svg`（或者把实拍图放入 `assets/img/`，再改各页 `<img src=...>`）
